sushiyuki-for-engineers
=======================

SUSHIYUKI emotion stickers for engineers!

寿司ゆきスタンプのエンジニア向けパックを作ります。

40個作成できたら LINE スタンプとして申請しますが、個々の素材は CC BY-NC-ND 4.0 で利用可能にします。
